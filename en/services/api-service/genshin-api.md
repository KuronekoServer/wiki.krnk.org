# Genshin API
## Overview
This API returns restructured data from Enka.Network in JSON format for Genshin user characters.

### Example API
```
https://genshin-api.krnk.org/api/genshindata/?uid=UID&scoretype=SCORE&charaName=CHARACTER
```

Service page: https://genshin-api.krnk.org

> If you make more than 100 requests in 10 seconds, you'll receive 429 Too Many Requests.
