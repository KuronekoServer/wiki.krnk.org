# Artifacter
## About
Artifacter was created as an official successor bot after the end of Genshin Artifacter. It is now operated under contract from Hyugo, who led Genshin Artifacter.

## Commands and Guide
Command | Description
-- | --
/build | Generate a Genshin build card.
/build-hsr | Generate a Honkai: Star Rail build card.
/party | Generate a Genshin party build card.
/help | Show command descriptions and change/delete your UID.

## FAQ
> What is the number at the top of the image?

The number helps identify the time and internal ID when an image issue occurs. It contains no personally identifiable information.

> Why do images look similar to other bots?

The image generation code is published (https://github.com/FuroBath/ArtifacterImageGen) and multiple developers make bots based on it.
