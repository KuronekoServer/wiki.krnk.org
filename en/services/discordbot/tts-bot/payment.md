# Paid Plans
## Overview
Paid plans relax the dictionary limits and reading length limits used by the TTS BOT.
Changing limits does not update existing settings automatically, so please adjust them again with the related commands when needed.

## Related Commands
Command | Description | Guide
-- | -- | --
[/subscription info](subscription.md) | Check the current plan and limits. | [/subscription](subscription.md)
[/subscription register](subscription.md) | Register a subscription code. | [/subscription](subscription.md)
[/subscription unregister](subscription.md) | Remove the current subscription code. | [/subscription](subscription.md)
[/server-settings length-limit](server-settings.md) | Change the message reading character limit. | [/server-settings](server-settings.md)
[/dictionary add](dictionary.md) | Dictionary entry limits and regex support depend on the plan. | [/dictionary](dictionary.md)
