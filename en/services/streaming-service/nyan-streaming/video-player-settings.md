# Video Player Settings
When using AVPro, enable the "Low Latency" option in the AVPro VideoPlayer component's Stream Mode.
The list below includes players verified by KuronekoServer staff and users.

## YamaPlayer
YamaPlayer supports low latency by default; no configuration changes are required.
![YamaPlayer](https://cdn.krnk.org/kuronekoserver/wiki/nyst/yamaplayer.webp)

BOOTH: https://booth.pm/ja/items/4768863

## iwaSync3
For low-latency viewing with iwaSync, change player settings in Unity and enable the options shown below.
![iwaSync3](https://cdn.krnk.org/kuronekoserver/wiki/nyst/iwasync.webp)

You may consider resync tools useful for VJs.

- BOOTH: https://nmxi.booth.pm/items/4883377
- BOOTH: https://booth.pm/ja/items/7423970

## VizVid
For VizVid, enable the settings shown in the screenshot below for low-latency playback.
![VizVid](https://cdn.krnk.org/kuronekoserver/wiki/nyst/vizvid.webp)

## ProTV 3.0
ProTV 3.0 supports low latency by default; no changes required.
