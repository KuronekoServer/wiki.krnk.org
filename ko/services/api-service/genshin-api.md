# 원신 API
## 개요
이 API는 Enka.Network의 데이터를 재구성하여 JSON 형식으로 원신 사용자 캐릭터 정보를 반환합니다.

예시 API:
```
https://genshin-api.krnk.org/api/genshindata/?uid=UID&scoretype=SCORE&charaName=CHARACTER
```

서비스 페이지: https://genshin-api.krnk.org

주의: 10초간 100회 이상 요청 시 429 Too Many Requests가 반환됩니다.
