# 접속 명령어
## 개요
읽어주기 BOT을 음성 채널에 참가시키거나, 연결을 끊거나, 현재 재생 중인 읽기 큐를 건너뛸 때 사용하는 기본 명령어입니다.

## 명령어 목록
명령어 | 설명 | 주요 인수
-- | -- | --
/join | 음성 채널에 참가합니다. | 음성 채널, 강제
/disconnect | 음성 채널에서 연결을 해제합니다. | 없음
/skip | 현재 읽기를 건너뛰고 큐를 비웁니다. | 없음

## /join
음성 채널에 참가합니다.
음성 채널 인수를 생략하면, 명령어 실행자가 현재 참가 중인 음성 채널에 접속합니다.

```text
/join
/join 음성 채널:#잡담VC
/join 음성 채널:#잡담VC 강제:true
```

참고:
- 실행자가 접속할 수 없는 음성 채널에는 참가할 수 없습니다.
- 이미 같은 음성 채널에 접속해 있는 경우 현재 텍스트 채널이 새 읽기 대상으로 추가됩니다.
- 다른 음성 채널에서 사용 중일 때 `강제`를 지정하지 않으면 이동 확인이 표시됩니다.

## /disconnect
현재 접속 중인 음성 채널에서 연결을 해제합니다.

```text
/disconnect
```

참고:
- BOT이 접속 중일 때만 사용할 수 있습니다.
- 현재 BOT이 접속한 음성 채널을 제어할 수 있는 사용자가 실행할 수 있습니다.

## /skip
현재 읽는 내용을 건너뛰고 대기 중인 큐를 비웁니다.

```text
/skip
```

참고:
- BOT이 접속 중일 때만 사용할 수 있습니다.
- 현재 BOT이 접속한 음성 채널을 제어할 수 있는 사용자가 실행할 수 있습니다.
