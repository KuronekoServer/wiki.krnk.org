# 유틸리티 명령어
## 개요
BOT 안내, 응답 확인, 초대 링크 확인에 사용하는 명령어입니다.

## 명령어 목록
명령어 | 설명 | 참고
-- | -- | --
/help | BOT 소개와 명령어 목록을 표시합니다. | 일부 환경에서는 구독 경고도 함께 표시됩니다.
/ping | BOT 응답 속도를 확인합니다. | 대략적인 응답 시간을 밀리초로 표시합니다.
/exvoice | 사용 가능한 ExVoice 목록 URL을 표시합니다. | 응답은 본인에게만 표시됩니다.
/invite | 초대 링크를 표시합니다. | 응답은 본인에게만 표시됩니다.

## /help
BOT 소개와 사용 가능한 슬래시 명령어를 표시합니다.

```text
/help
```

## /ping
BOT 응답 속도를 확인합니다.

```text
/ping
```

## /exvoice
사용 가능한 ExVoice 목록 URL을 표시합니다.

```text
/exvoice
```

## /invite
BOT 초대 링크를 표시합니다.

```text
/invite
```
