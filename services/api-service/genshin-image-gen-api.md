# Artifacterビルドカード生成 API
## 概要
Enka.NetworkのAPI情報を使用し、Artifacterのデザインを使用した画像生成APIとなっております。

GETリクエストでAPIにアクセスし、様々なビルドカードの画像を生成できるAPIとなっています。


# API仕様

https://artifacter-imagegen-api.krnk.org/docs

サービスページ: https://artifacter-imagegen-api.krnk.org

> 1秒間に10回以上リクエストをすると「429 Too many Requests」が返されます。
APIのレートリミットの緩和を行いたい場合は[お問い合わせ](https://discord.krnk.org)をお願いします。
