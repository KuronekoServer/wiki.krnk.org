# Utility Commands
## Overview
These commands are used for bot guidance, response checks, and invite links.

## Commands
Command | Description | Notes
-- | -- | --
/help | Show the bot overview and command list. | In some environments it also shows subscription warnings.
/ping | Check the bot response time. | Shows an approximate response time in milliseconds.
/exvoice | Show the URL for the available ExVoice list. | The response is shown only to you.
/invite | Show the invite link. | The response is shown only to you.

## /help
Show the bot overview and the available slash commands.

```text
/help
```

## /ping
Check the bot response time.

```text
/ping
```

## /exvoice
Show the URL for the available ExVoice list.

```text
/exvoice
```

## /invite
Show the bot invite link.

```text
/invite
```
