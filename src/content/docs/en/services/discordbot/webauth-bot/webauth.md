# /webauth
## Overview
Set the role used for authentication and install the authentication panel.

![image](https://cdn.krnk.org/kuronekoserver/wiki/webauth-bot/webauth.webp)

## Usage
```
/webauth <option>
```

Option | Description | Required
--- | --- | ---
role | Set the role | yes
