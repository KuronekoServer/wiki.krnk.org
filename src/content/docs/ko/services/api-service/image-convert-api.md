# 이미지 변환 API
## 개요
이 API는 이미지를 받아 다양한 형식으로 변환하여 응답합니다.

단일 이미지가 업로드된 경우, 변환된 WebP 이미지가 직접 응답됩니다.

여러 이미지가 업로드된 경우, 변환된 이미지가 포함된 ZIP 파일이 응답됩니다.

### 문서

https://image-convert-api.krnk.org/docs

#### 주의사항

- 파일 크기에 따라 처리에 시간이 걸릴 수 있습니다.
- 파일 확장자가 이미지 형식을 지원해야 합니다.


:::danger
1초에 50회 이상 요청하면 "429 Too many Requests"가 반환됩니다.
API 레이트 리밋 완화를 원하시면 [문의](https://discord.krnk.org)해 주세요.
:::
