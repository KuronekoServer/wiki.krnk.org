# Nyan Streaming
### 서비스 개요
본 서비스는 고화질 영상을 1초 이내의 지연으로 송출하는 것을 목표로 하는 영상 송출 서비스입니다.
※HLS로 시청할 때의 지연은 예외로 합니다.

https://nyst.live/
