# ユーティリティコマンド
## 概要
BOTの案内表示や応答確認、招待リンクの確認に使うコマンドです。

## コマンド一覧
コマンド名 | 概要 | 補足
-- | -- | --
/help | BOTの説明とコマンド一覧を表示します。 | 一部環境では、サブスクリプション未加入時の案内も表示されます。
/ping | BOTの応答速度を確認します。 | おおよその応答時間をミリ秒で表示します。
/exvoice | 利用可能なExVoice一覧のURLを表示します。 | 返信は自分だけに表示されます。
/invite | 招待リンクを表示します。 | 返信は自分だけに表示されます。

## /help
BOTの概要と、利用できるスラッシュコマンドの一覧を表示します。

```text
/help
```

## /ping
BOTの応答速度を確認します。

```text
/ping
```

## /dashboard
サーバーのダッシュボードのURLを表示します。

```text
/dashboard
```

## /exvoice
利用可能なExVoice一覧へのURLを表示します。

```text
/exvoice
```

## /invite
BOTの招待リンクを表示します。

```text
/invite
```
